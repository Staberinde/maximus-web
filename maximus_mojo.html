NAME

maximus_mojo.pl - Mojolicious::Lite app

SYNOPSIS

maximus_mojo.pl [options]

DESCRIPTION

This Mojolicious::Lite app can be used to run a lightweight service for serving module packages from the database.

HELPERS

cfg

Loads the maximus.conf file using Config::Any. Which file to load can be modified by setting the MAXIMUS_CONFIG_LOCAL_SUFFIX environment variable.

db

Helper for setting up the database connection. Returns a Maximus::Schema object.

ROUTES

/module/download/:scope/:modules/#version

Serve module file from database or remote_location if it has been set. Route matches that of the Catalyst application.

ENVIRONMENT

Set MAXIMUS_CONFIG_LOCAL_SUFFIX to specify which config file to be additonally loaded. Set it to testing to load maximus_testing.conf.
