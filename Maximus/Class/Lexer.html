NAME

Maximus::Class::Lexer - BlitzMax Lexer

SYNOPSIS

use Maximus::Class::Lexer;
my $lexer = Maximus::Class::Lexer->new;
my @tokens = $lexer->tokens('string');

DESCRIPTION

Provides a minimal BlitzMax lexer to retrieve information about dependant modules and which files the module depends on.

TOKENS

COMMENT

BlitzMax comments start with ' for single lines or multiple lines between Rem and End Rem keywords.

MODULEDESCRIPTION

The module description from the bbdoc: part above the Module declaration.

MODULENAME

The module nam from the bbdoc: part above the Module declaration.

MODULEVERSION

The module version number from a ModuleInfo line.

MODULEAUTHOR

The module author(s) from a ModuleInfo line.

MODULELICENSE

The module license from a ModuleInfo line.

HISTORY

The module history from a ModuleInfo line.

DEPENDENCY

Dependencies are found with Import and Framework keywords.

INCLUDE_FILE

Included files are found with the Include and Import keywords when the included or imported file ends with .bmx.

METHODS

tokens($input_iterator)

Scan input and return a list with tokens
