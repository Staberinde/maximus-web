NAME

Maximus::Class::Module::Source::SCM::Git - Handles module sources inside a Git repository

SYNOPSIS

use Maximus::Class::Module::Source::SCM::Git;
my $source = Maximus::Class::Module::Source::SCM::Git->new;

DESCRIPTION

Git support for retrieving the modules sources.

ATTRIBUTES

repository

Location of remote Git repository. Must be publicly readable

local_repository

Location of the local copy of the Git repository

tags_filter

If the repository hosts more modules then set this to filter the listing. e.g.: ^v(.+) if this module uses tags in the style of v0.01 or v0.3.0. You MUST add a capture so the version string can be fetched.

main_branch

Defaults to master, change it to use a different branch.

METHODS

init_repo

Initialize repository. Either clones or pulls to update

prepare

Fetch files for version and store them inside the temporary directory

get_versions

Returns all versions

get_latest_revision

Retrieve latest revision of main_branch

auto_discover

Discover available modules from the repository

apply_scm_settings

Apply SCM specific settings
