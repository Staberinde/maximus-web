NAME

Maximus::Class::Module::Source::SCM::Subversion - Handles module sources inside a Subversion repository

SYNOPSIS

use Maximus::Class::Module::Source::SCM::Subversion;
my $source = Maximus::Class::Module::Source::SCM::Subversion->new;

DESCRIPTION

Subversion support for retrieving the modules sources.

ATTRIBUTES

repository

Location of remote Subversion repository. Must be publicly readable

local_repository

Location of the local copy of the Subversion repository

trunk

Path to trunk. If the repository hosts more modules then set it to the module path, e.g. trunk/my.mod

tags

Path to tags. If the repository hosts more modules then set tags_filter to filter the listing.

tags_filter

If the repository hosts more modules then set this to filter the listing. e.g.: ^my\.mod-(.+) if this module uses tags in the style of my.mod-0.01 or my.mod-0.3.0. You MUST add a capture so the version string can be fetched.

apply_scm_settings

Apply SCM specific settings

METHODS

init_repo

Initialize repository. Either clones or pulls to update

prepare

Fetch files for version and sort them inside the temporary directory

get_versions

Returns all versions

get_latest_revision

Retrieve latest revision of trunk

auto_discover

Discover available modules from the repository
