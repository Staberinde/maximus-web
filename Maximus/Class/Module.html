NAME

Maximus::Class::Module - Represents a module

SYNOPSIS

use Maximus::Class::Module;
my $module = Maximus::Class::Module->new;

DESCRIPTION

This class represents a module

ATTRIBUTES

modscope

Modscope (namespace) of module, e.g. brl.example

mod

Name of module, e.g. brl.example

desc

Description of module

source

Source location. Needs to be a class which does Maximus::Role::Module::Source

scm_settings

SCM specific settings

schema

"DBIx::Class schema"

announcer

A Maximus::Broadcast::Announcer object. If available will broadcast a module update when saved.

METHODS

save($user)

Save module in database. $user should be a DBIx::Class::Row from Maximus::Schema::Result::User.

When no source has been given when constructing the object this method returns the DBIx::Class::Row that contains the record of the module.

If a announcer object has been given it will broadcast the module release to the different drivers, but only if it's not a dev release and source has been given as well.
