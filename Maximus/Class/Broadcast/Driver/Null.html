NAME

Maximus::Class::Broadcast::Driver::Null - Test driver

SYNOPSIS

use Maximus::Class::Broadcast::Driver::Null;
my $driver = Maximus::Class::Broadcast::Driver::Null->new;
my $msg = Maximus::Class::Broadcast::Message->new(text => 'Hello world!');
$driver->say($msg);

DESCRIPTION

This driver is mainly for testing purposes. Any message passed to it will be ignored. It does however count the number of times say has been called.

ATTRIBUTES

counter

Contains the number of times say has been called.

METHODS

say(Maximus::Class::Broadcast::Message $msg)

Ignores the message but adds 1 to the counter attribute.
