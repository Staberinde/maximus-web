NAME

Maximus::Controller::Timeline - Timeline controller

DESCRIPTION

This controller is responsible for serving the timeline/announcements of the website.

METHODS

base

Collects all announcements and stores them in the stash as announcements. This is use by index, atom and rss.

index

Normal HTML view of the timeline.

atom

Generates an Atom feed.

rss

Generates a RSS feed.

SEEO ALSO

Maximus::View::Feed
