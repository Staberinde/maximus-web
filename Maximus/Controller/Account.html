NAME

Maximus::Controller::Account - Catalyst Controller

DESCRIPTION

Catalyst Controller to manage accounts

METHODS

index

My Account page, only visible when logged in. Redirects to login when not logged in.

login

Login to get access to My Account.

logout

Logout a logged in account.

signup

Signing up a new account. If recaptcha is configured it'll use the reCAPTCHA service to prevent bots from signing up.

forgot_password

Allows a user to reset an account's password.

reset_password

Expects a username and a hash. If the hash is faulty nothing will happen. After this action has been succesfully executed the link will be expired because the password has been changed.

edit

Allow for the user to edit its account details. If the user doesn't have the role user-id-mutable it will display a 403 forbidden error.
